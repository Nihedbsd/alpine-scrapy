# hmmbug/alpine-scrapy

Python scrapy in an Alpine docker container Edit

- Included libraries:
 - libxslt libxml2 jpeg tiff libpng zlib git curl libpq
- Included python libraries:
 - scrapy
 - scrapyd
 - scrapyd-client
 - scrapy-splash
 - mysql-connector-python
 - simplejson
 - elasticsearch & elasticsearch_dsl
 - pillow
 - psycopg2

- /var/log/scrapy
- /var/lib/scrapy

# Ports

- 6800
